# Database Uitleg

## H2 Console Gebruiken

### Stap 1: Start je Spring Boot applicatie
Zorg dat je `backend` applicatie draait.

### Stap 2: Open de H2 Console
Ga naar: **http://localhost:8081/h2-console**


### Stap 3: Verbind met de database

**JDBC URL:** 
`jdbc:h2:file:./database-db` 

**Gebruikersnaam:** `sa`  
**Wachtwoord:** (leeg laten)
